# Мемератор 3000 - Генератор мемных фраз

(c) 2025, Киберпанк Лаборатория

Для всех, кто после 30, но душой всё ещё в мемах.

## Описание

Мемератор 3000 - это киберпанковый генератор мемных фраз, созданный специально для молодежи после 30 лет. Программа генерирует уютный юмор в стиле киберпанка, подходящий для вывода в консоли по команде.

## Особенности

- Современный киберпанковый стиль
- Уютный юмор без жести
- Ориентация на аудиторию после 30 лет
- Генерация случайных мемных фраз
- Поддержка вывода нескольких фраз за раз
- Возможность просмотра всех доступных мемов

## Требования

- Python 3.6 или выше

## Установка

Скачайте файл `memerator.py` и запустите его с помощью Python:

```bash
py memerator.py
```

## Использование

### Генерация одной мемной фразы

```bash
py memerator.py
```

### Генерация нескольких фраз

```bash
py memerator.py -n 3
```

или

```bash
py memerator.py --batch 5
```

### Просмотр всех доступных мемов

```bash
py memerator.py --list
```

### Просмотр справки

```bash
py memerator.py --help
```

## Примеры вывода

```

```

```
[КОД:01] [09:41:21] Я не торможу, я просто в режиме энергосбережения как старый процессор
[КОД:01] [09:41:21] Мои мемы - это как старый добрый Python - простой, но мощный
[РЕЗЕРВ:55] [09:41:21] У меня не возрастное кризис, у меня обновление системы
```

## Стиль

Все мемные фразы выполнены в современном киберпанковом стиле с использованием технических метафор и юмора, понятного людям, которые выросли в эпоху цифровых технологий. Фразы содержат уютный юмор, направленный на самопринятие и ироничное отношение к возрасту.

## Целевая аудитория

Молодежь после 30 лет, которая:
- Работает в IT или связана с цифровыми технологиями
- Понимает киберпанковую эстетику
- Ценит уютный юмор без жесткого сарказма
- Хочет посмеяться над своими возрастными особенностями
